2. 课堂小作业1 ---**task1_1.slx**  
好像就是把老师给的几个公式用simulink搭一下，有两个阶跃响应函数，一个稍微简单，一个有点复杂，二者动态品质差不多的。简单的对应的控制器传递函数不是真有理函数，复杂的对应控制器的传递函数不仅稳定真有理，而且简单，工程易实现。但是实际设计过程中，设计人可能想不到复杂的，从而说明控制系统设计的难点所在。
